# 🌸 Iris Flower Classification

This is a simple Machine Learning project that classifies iris flowers into three species:

- Setosa
- Versicolor
- Virginica

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## 🧠 Model
Logistic Regression is used for classification.

## 📊 Dataset
Built-in Iris dataset from sklearn.
